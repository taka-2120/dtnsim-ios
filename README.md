# dtnsim for iOS

## Requirements

**Run**

- iOS 18 or later

**Develop**

- Swift 6 or later
- Xcode 16 or later

## Description

This is a DTN (Delay/Disruption Tolerant Networking) simulator for iOS. This project was inspired by [h-ohsaki/dtnsim](https://github.com/h-ohsaki/dtnsim).
